Welcome to the LYNX Real Estate Group.

This isn’t just onboarding — it’s the system we use every day to work with clients, run deals, and build a real estate business that actually lasts.
:::

---

## How to use this guide

This is not meant to be skimmed.

Go through it **in order** during your first month, then come back to it whenever you need clarity on how something is done here.

Everything inside is built around:
- real conversations
- real workflows
- real expectations

If something isn’t clear — ask.  
If something feels off — bring it up.  
You’re not expected to figure this out alone.

---

## What you’ll learn

This guide is broken into five core modules:

### Module 1 — Welcome to the Team
How we operate, what we expect, and how to set yourself up properly from day one.

### Module 2 — Your Tech Stack
The tools you’ll use daily — FUB, MLS, showings, communication, and documents.

### Module 3 — Working With Buyers
Lead calls, consultations, showings, offers, and getting clients to the finish line.

### Module 4 — Working With Sellers
Listings, communication, pricing strategy, and managing sellers professionally.

### Module 5 — Building Your Business
Lead generation, follow-up, referrals, and long-term growth.

## The standard here

:::warning
The agents who succeed here do three things consistently:

- Show up prepared  
- Follow the system  
- Actually care about the client  

That’s it.
:::

You don’t need to be perfect.  
You do need to be consistent.
